# Cognizant-DN5

Welcome to the Cognizant Digital Nurture 5.0 (DN5) repository — a single workspace combining the Upskilling and Deep Skilling learning tracks.

## Cognizant Digital Nurture 5.0 Journey
This repository captures the learning path, hands-on exercises, assignments, and mini-projects for participants of the DN5 program. It is organized to help learners track progress, store notes, and maintain reusable examples.

## Upskilling Track
The `DN5-Upskilling` folder contains foundational topics such as HTML5, CSS3, JavaScript, SQL, and C# with notes, hands-on labs, and assignments to practice core skills.

## Deep Skilling Track
The `DN5-DeepSkilling` folder contains advanced topics including Design Patterns, DSA, Advanced SQL, Entity Framework Core, ASP.NET Core Web API, Microservices, Angular, Docker, and Git-CICD.

## Technologies Covered
- Web: HTML5, CSS3, JavaScript, Angular
- Backend: ASP.NET Core, Entity Framework Core, Microservices
- Data: SQL, Advanced SQL
- DevOps: Docker, Git, CI/CD
- Patterns & Algorithms: Design Patterns, DSA

## Progress Tracker
Each topic includes a `Notes.md`, a `HandsOn` folder, and an `Assignments` folder. Use the `Notes.md` to record learning summaries and the `HandsOn` and `Assignments` folders for exercises. Mark completed items in your personal notes.

---

This repository is intended to be a single Git repository at the root (one `.git` folder). Initialize locally, commit, then push to your GitHub account to publish the repository.
